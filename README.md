lived
=====

easy to use reactive editor to your text area.

demo
====

check this out: http://automata.github.com/lived/demo.html

============

* jquery
* jquery UI (for sliders)
* jquery color picker plugin spectrum http://bgrins.github.com/spectrum/
* codemirror
